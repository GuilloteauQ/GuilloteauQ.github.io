Langage C

Histoire

  • inventé dans les années 1970

  • beaucoup de domaines d’applications (Systèmes d’exploitations, Embarqué, HPC)

  • a influencé beaucoup de langages par la suite (C++, Java, Python, Rust…)

Terminologie

  • Code source: code en langage C (la recette lisible par l’humain)

  • Assembleur: Langage machine (un dialecte obscur)

  • Instruction: opération assembleur (une étape de la recette dans le dialecte)

  • Compilateur: transforme du code C en assembleur (une boite noire qui traduit notre recette pour la machine)

  • Binaire: forme exécutable du programme (notre gâteau cuisiné par la machine)

Boucles

Motivation

Imaginez, vous avec un tableau de taille 100 et vous faire la somme de ses valeurs.

int tableau[100] = { /* toutes les valeurs */ };
int somme = 0;
somme += tableau[0];
somme += tableau[1];
// ....
somme += tableau[99];

printf("La somme vaut: %d\n", somme);

BERK!!!

Boucle “Pour” - Exemple

int tableau[100] = { /* toutes les valeurs */ };
int somme = 0;
for (int i = 0; i < 100; i+=1) {
  somme += tableau[i];  
}
printf("La somme vaut: %d\n", somme);

Beaucoup plus propre !

Boucle “Pour” - Définition

for (<VARIABLE>, <CONDITION FIN>, <OPERATION>) {
  <BLOC CODE>
}

  • <VARIABLE>: initialisation du compteur de la boucle

  • <CONDITION FIN>: la condition pour terminer la boucle

  • <OPERATION>: comment changer le compteur à chaque tour de boucle

Exercice

Ecrire une boucle qui calcule la factorielle de 10.

\(10! = 10 \times 9 \times 8 \times ... \times 2 \times 1\)

Solution

int fact = 1;
for (int i = 1; i <= 10; i += 1) {
  fact = fact * i;
}
printf("10! = %d\n", fact);

Exercice

Ecrire une boucle while qui calcule la suite de Collatz pour un entier.

\[ C_n = \begin{cases} \frac{C_{n-1}}{2} & \text{si } C_{n-1} \text{ est pair}\\ 3 \times C_{n-1} + 1 & \text{sinon} \end{cases} \]

La suite est terminée lorsque \(C_n = 1\)

Solution

  printf("Cn = %d\n", c_n);

    c_n = c_n / 2;
  } else {
    c_n = 3 * c_n + 1;
  }
}

Fonction

Définition

Fonction: Bloc de code autocontenu qui effectue une tâche spécifique.

<TYPE> ma_fonction(<ARGUMENTS>) {
  // Quelque chose ici
  return <QQCHOSE>;
}

  • <TYPE>: type d’objet retourné

  • <ARGUMENTS>: liste de paramètres et leurs types

  • <QQCHOSE>: objet retourné, de type <TYPE>

Exemple

Fonction retournant le carre d’un entier:

int carre(int x) {
  int x_carre = x * x;
  return x_carre;
}

aussi possible

int carre(int x) {
  return x * x;
}

Appeler une fonction

int x = 4;
int x_carre = carre(x);

Plusieurs paramètres

int aire_rectangle(int longueur, int largeur) {
  int aire = longeur * largeur;
  return aire;
}
int a = 2;
int b = 4;
int aire = aire_rectangle(a, b);

Scope

int add(int a, int b) {
  return a + b;  
}

int main() {
  int a = 10;
  int c = add(1, 2);
  printf("c = %d\n", c); // c = ??? 
};

Structure d’un programme

Point d’entrée

// des trucs ici

int main(int argc, char** argv) {
  // et ici
  return 0;
}

// ou encore là

  • main = point d’entrée

  • argc = nb d’arguments

  • argv = les arguments

Ex: ./mon_prgm 1 bob 3.14

argc = ? argv = ?

Arguments lignes de commande

  • argc = 4

  • argv = {"./mon_prgm", "1", "bob", "3.14"}

  • atoi, atof: converti des chaines de caractères en entiers ou flottants.

Exercice

Ecrire un fonction qui calcule une puissance d’un entier

\[ x^y \]

Solution

int puissance(int x, int y) {
  if (y == 0) {
    return 1;
  }
  int p = x;
  for (int i = 1; i < y, i+=1) {
    p = p * x;
  }
  return p;
}

Exercice

    return 1;
  }
  return x * puissance(x, y - 1);
